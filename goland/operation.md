# Main Menu
## File
1. Open => `Command` + `Option` + `1`, `O`
2. New
    * Directory => `Command` + `Option` + `1`, `D`

## Refactor
1. Rename => `Command` + `Option` + `6`, `M`

# Editor
## Folding
1. Expand => `Command` + `+`
2. Collapse => `Command` + `-`

## Word Related
1. Move Caret To Next Word => `ALT` + `->`
2. Delete Next Word => `Alt` + `DEL`
